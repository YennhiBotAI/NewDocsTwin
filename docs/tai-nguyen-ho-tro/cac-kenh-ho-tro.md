## 📝 Preparing Your Support Request

### Required Information
```
📧 Subject: [Category] Brief description
🏷️ Plan: Free/Pro/Enterprise
👤 Account: your-email@domain.com
🔍 Issue: Detailed description
📅 When: When did this start?
🔄 Frequency: How often does this happen?
📱 Environment: Browser, OS, device
```

### For Technical Issues
```
🐛 Error messages: Copy exact text
📸 Screenshots: What you're seeing
🔗 URLs: Specific pages with issues
🔧 Steps: How to reproduce the problem
🌐 Browser: Chrome/Safari/Firefox + version
💻 OS: Windows/Mac/Linux + version
```

### For API Issues
```
🔑 API Key: Last 4 characters only
📍 Endpoint: Which API endpoint
📊 Request: Sample request payload
📄 Response: Error response received
📈 Rate Limits: Current usage status
🕐 Timestamp: When did error occur
```

## 💡 Self-Help Tips

### Before Contacting Support
1. **Search FAQ**: 80% of questions already answered
2. **Check Status Page**: Might be a known issue
3. **Try incognito mode**: Rules out browser issues
4. **Clear cache**: Fixes many UI problems
5. **Update browser**: Ensure latest version

### Speed Up Resolution
1. **Provide screenshots**: A picture = 1000 words
2. **Include error messages**: Copy exact text
3. **List reproduction steps**: Help us recreate issue
4. **Mention urgency level**: Helps us prioritize
5. **Stay available**: For follow-up questions

### Community Etiquette
1. **Search first**: Avoid duplicate questions
2. **Use clear titles**: Make it easy to find
3. **Provide context**: Include relevant details
4. **Thank helpers**: Appreciate community effort
5. **Share solutions**: Help others with same issue
